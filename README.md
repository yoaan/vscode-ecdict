# vscode-ecdict

ECDICT extension for coc.nvim, fork of [coc-ecdict](https://github.com/fannheyward/coc-ecdict)

## Usage

安装后在第一次使用时会下载 ECDICT，下载完即可使用悬浮弹窗翻译。由于是使用本地词库，所以速度很快。
